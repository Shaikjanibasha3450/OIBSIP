# OASIS INFOBYTE SIP - All Tasks

This repository contains all the tasks completed during the OASIS INFOBYTE Summer Internship Program (SIP).

## Task Overview

### Task 1 - Internship
General internship-related materials and documentation.

### Task 2 - Unemployment Analysis
Data analysis project focusing on unemployment patterns and trends.

### Task 3 - Car Price Prediction
Machine learning project to predict car prices using regression models.

### Task 4 - Spam Email Detection with ML
Machine learning project to detect and classify spam emails using various ML algorithms.

### Task 5 - Sales Prediction Using Python
Data science project to predict sales using Python and machine learning techniques.
